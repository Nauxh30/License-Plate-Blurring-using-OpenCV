# License-Plate-Blurring-using-OpenCV


## 📌 Project Overview
This project demonstrates Object Detection using OpenCV and Haar Cascade Classifiers.  
The system detects Russian vehicle license plates from an image and applies blur to hide sensitive information.

---

# ✨ Features
- Detects license plates in images
- Applies Gaussian Blur to detected plates
- Uses Haar Cascade XML classifier
- Beginner-friendly Computer Vision project
- Implemented in Python using OpenCV

---

# 🛠️ Technologies Used
- Python
- OpenCV
- NumPy
- Matplotlib
